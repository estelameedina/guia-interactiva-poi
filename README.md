# Guía Interactiva de Puntos de Interés

## Descripción

Aplicación web que permite explorar puntos de interés (POI) cercanos y reproducir contenido multimedia asociado.

Incluye funcionalidad de geolocalización para detectar proximidad del usuario a los distintos puntos.

---

## Funcionalidades

- Listado dinámico de puntos de interés
- Reproducción de audio con HTML5
- Detección de ubicación del usuario
- Aviso de proximidad a lugares cercanos

---

## Tecnologías

- HTML5 (audio)
- CSS3
- JavaScript (DOM + Geolocation API)

---

## Uso

1. Clonar el repositorio
2. Abrir con Live Server (recomendado)
3. Permitir acceso a la ubicación

---

## Privacidad

La aplicación solicita permisos de ubicación, pero no almacena ni envía datos personales.
